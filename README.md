# E-Commerce Sales Analysis Dashboard

## Project Overview

This project presents a comprehensive analysis of an e-commerce sales dataset using Microsoft Excel. The objective was to transform raw transactional data into actionable business insights through data cleaning, KPI development, exploratory analysis, and dashboard visualization.

## Business Objective

The analysis focuses on evaluating overall business performance, identifying high-performing product categories, assessing regional profitability, understanding customer purchasing behavior, and monitoring sales trends over time. The resulting dashboard provides a centralized view of key performance indicators to support data-driven decision-making.

## Tools and Techniques

Microsoft Excel was used for data cleaning, data preparation, KPI calculation, Pivot Table analysis, Pivot Chart creation, and dashboard development.

## Key Performance Indicators

The dashboard highlights critical business metrics, including Total Sales, Total Profit, Total Orders, and Average Order Value, providing a high-level overview of organizational performance.

## Analysis Conducted

The project includes category-level sales analysis, regional profit analysis, customer performance evaluation, product performance assessment, and monthly sales trend analysis. These analyses were performed using Pivot Tables and visualized through interactive charts to uncover meaningful business insights.

## Dashboard

The final dashboard consolidates all major metrics and visualizations into a single view, enabling stakeholders to quickly assess business performance and identify growth opportunities.
